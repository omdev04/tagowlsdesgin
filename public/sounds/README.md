Place your custom meeting ringtone file in this folder.

Default expected file path:
- /sounds/meet-ringtone.mp3

You can also use a custom URL/path via env:
- NEXT_PUBLIC_MEET_RINGTONE_URL=/sounds/your-file-name.mp3

Notes:
- Prefer short ringtone clips (1-5 seconds).
- Supported browser formats are typically mp3, wav, or ogg.
